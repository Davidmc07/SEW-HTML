Código

A continuación se muestran varios ejemplos de código escrito en rust mediante los cuales es posible hacerse una idea de la apariencia y el funcionamiento de este lenguaje de programación.

Hola mundo


fn main() {

}

Declaración de variables

Las variables en rust son inmutables por defecto. Para poder modificarlas deberemos añadir mut en su declaración.


fn main() {
    let inmutable = 5;   // variable inmutable
    let mut mutable = 5; // variable mutable
}

Entrada de datos

Para leer datos del usuario necesitamos la biblioteca io, por lo que deberemos añadir use std::io al principio de nuestro código.


use std::io;

fn main() {
    let mut str = String::new();

    println!("Escribe una línea");

    io::stdin().read_line(&mut str)
        .ok()
        .expect("Fallo al leer linea");
                
    println!("Has escrito '{}'", str);
}

Ciclos

En rust, al igual que en otros lenguajes de programavión, tenemos ciclos for, for each y while.

Ciclos for


fn main() {
    for i in 0..10 {
        println!("Iteración número {}", i);
    }
}

Ciclos for each


fn main() {
    let mut my_arr: Vec<String> = Vec::new();
                
    my_arr.push("String1".to_string());
    my_arr.push("String2".to_string());
    my_arr.push("String3".to_string());
                
    for word in my_arr {
        println!("{}", word);
    }
}

Ciclos while


fn main() {
    let mut n = 0;
    while n < 11 {
        println!("Iteración número {}", i);
        n += 1;
    }
}
